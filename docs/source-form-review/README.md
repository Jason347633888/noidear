## 每张表只做一个判断

| 判断 | 含义 |
|---|---|
| 保留独立填写 | 这张表业务上要单独填 |
| 同类共用页面 | 多张表本质一样，只是产品、区域、设备、项目不同 |
| 自动生成或台账视图 | 不手工填第二套，从已有数据打印或汇总 |
| 只当附件 | 文件、证书、扫描件、受控文件原件 |
| 暂缓或不要做 | 现在不需要，或已有系统覆盖 |

复核时不用写技术词，直接说：

```text
第 1 组：同意
第 2 组：第 35 张要单独填
第 3 组：清洁表先全部保留独立填写
```

## 已确认的总原则

- 后面没有动态表单模块。
- 每张仍在使用的表，要有可填写入口。
- 有填写入口，不等于每张表都做复杂独立功能。
- 重复表可以共用一个页面家族。
- 主数据要选已有产品、物料、供应商、员工、设备、区域。
- 历史记录要保存当时看到的快照，不能因为以后主数据改了而变样。
- 制造早期投料、配料、过程参数时，可能还没有成品批次；先按日期、时间、线别、产品、工序、班次记录，后面再关联批次。
