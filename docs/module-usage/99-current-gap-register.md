## 级别定义

| 级别 | 定义 |
|---|---|
| P0 | 影响追溯、批次、放行、库存、审批闭环或主数据事实源，必须优先整改 |
| P1 | 影响业务一致性、数据质量、跨模块联动或统计准确性 |
| P2 | 入口、命名、展示、体验或管理员配置问题 |
